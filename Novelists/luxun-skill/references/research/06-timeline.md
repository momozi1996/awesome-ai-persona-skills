# 鲁迅完整时间线

> 来源：百度百科、维基百科等公开资料整理

## 一、少年时期（1881-1898）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1881年9月25日 | 出生于浙江绍兴城内东昌坊新台门周家，幼名阿张，学名周樟寿 | 中国现代文学奠基人的起点 |
| 1892年 | 入三味书屋从寿镜吾读书，课余影描图画，与章闰水（闰土原型）建立友谊 | 接受传统启蒙教育，接触民间生活 |
| 1893年 | 祖父周介孚因科场案下狱，父周伯宜重病，家产中落，全家避难于乡下 | 家道中落，初识世态炎凉，为日后创作积累素材 |
| 1896年 | 父亲去世，家境益艰，开始写日记 | 少年丧父，承担家庭重担，过早体会人生艰辛 |
| 1897年 | 家族分房，分给鲁迅的既差且小，拒绝签字遭叔辈斥责 | 深感世态炎凉，形成对封建家族的批判意识 |

## 二、南京求学（1898-1902）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1898年4月 | 入江南水师学堂，改名周树人 | 离开故乡，开始新式教育 |
| 1898年12月 | 被催促参加县考，中榜后以四弟患病为由不再参加府考 | 放弃科举道路，选择新学 |
| 1899年 | 转入江南陆师学堂附设矿务铁路学堂学开矿 | 接触西方科学技术 |
| 1899-1901年 | 阅读赫胥黎《天演论》，接触进化论思想；爱骑马运动，敢于和旗人子弟竞赛 | 接受新思想洗礼，形成"物竞天择"观念 |
| 1902年1月 | 矿路学堂毕业 | 完成国内新式教育 |

## 三、日本留学（1902-1909）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1902年3月 | 与厉绥之、钱均夫同赴日本公费留学 | 东渡日本，寻求救国之道 |
| 1902年4月 | 入弘文学院普通科江南班（日语学习速成班） | 开始系统学习日语 |
| 1903年 | 剪辫，课余喜读哲学与文艺之书，尤注意人性及国民性问题 | 表明革命态度，开始关注国民性改造 |
| 1904年4月 | 弘文学院结业 | 完成语言学习 |
| 1904年6月 | 祖父介孚公卒 | 家族长辈离世 |
| 1904年9月 | 入仙台医学专门学校（现日本东北大学），相识藤野严九郎 | 选择医学救国道路 |
| 1906年1月 | 课间观"日俄战争教育片"，深受刺激，决定弃医从文 | **人生转折点**：认识到精神麻木比身体病痛更可怕 |
| 1906年6月 | 将学籍列入"东京独逸语协会"所设德语学校 | 开始学习德语 |
| 1906年夏秋 | 被骗回国与朱安结婚，旋即复赴日本 | 封建包办婚姻的悲剧 |
| 1906年7月 | 从仙台回到东京，不再入学，专门从事文艺译著工作 | 正式转向文艺救国 |
| 1906-1908年 | 学习德语、俄语，从事翻译工作 | 广泛接触西方文学 |
| 1908年 | 从章太炎先生学习，加入光复会；与周作人译《域外小说集》 | 接受革命思想，开始文学翻译实践 |
| 1909年 | 《域外小说集》二册出版 | 中国现代翻译文学的先声 |

## 四、归国初期（1909-1917）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1909年8月 | 归国，任杭州、浙江两级师范学堂生理学和化学教员，兼任日本教员铃木珪寿的植物学翻译 | 开始教育生涯 |
| 1910年8月 | 任绍兴中学堂教员兼监学 | 回乡任教 |
| 1911年 | 写个人的第一篇小说文言小说《怀旧》 | 小说创作初试 |
| 1912年 | 临时政府成立于南京，应蔡元培之邀，任教育部社会教育司第一科科长；8月被任命为北京政府教育部佥事 | 进入民国政府教育系统 |
| 1912-1917年 | 大量抄古碑，辑录金石碑帖，校对古籍，研究佛教思想 | "沉默期"，潜心古籍研究，积累文化底蕴 |
| 1917年7月7日 | 因张勋复辟乱作，愤而离职；14日乱平即返部 | 对政治乱象的愤慨 |

## 五、新文化运动时期（1918-1923）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1918年1月 | 参加《新青年》改组，任编委 | 加入新文化运动核心阵营 |
| **1918年5月** | **以"鲁迅"为笔名发表《狂人日记》，中国现代文学史上第一篇用现代体式创作的白话短篇小说** | **中国现代文学的开端，奠定了鲁迅的文学地位** |
| 1920年 | 在北京大学、北京高等师范学校讲授中国小说史；6月读《共产党宣言》中文译本盛赞译者；9月发表小说《风波》 | 进入高校教学，思想进一步开放 |
| 1923年8月 | 小说集《呐喊》出版 | 第一部小说集，收录《狂人日记》《阿Q正传》等经典 |
| 1923年8月 | 与弟弟周作人分居，迁至西四塔胡同61号居住 | 兄弟失和，个人生活的重大变故 |
| 1923年12月 | 作《娜拉走后怎样》演讲，兼任女师大、世界语学校教师；《中国小说史略》上册出版 | 关注女性解放问题，学术著作问世 |

## 六、创作高峰期（1924-1927）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1924年7月 | 赴西安讲《中国小说的历史变迁》 | 传播新文学理念 |
| 1924年11月 | 《语丝》周刊出版，鲁迅在首期发表《论雷峰塔的倒掉》，成为《语丝》作家群主将之一 | 加入语丝社，扩大文学影响 |
| 1925年 | 支持"女师大风潮"，被教育总长章士钊免除佥事职务；向平政院状告章士钊 | 投身学生运动，与北洋政府对抗 |
| 1926年3月 | "三·一八惨案"发生 | 目睹政府暴行 |
| 1926年4月 | 作《死地》《记念刘和珍君》等抨击段祺瑞政府屠杀学生的罪行，遭追捕，避难于山本医院 | **民主战士的姿态，以笔为枪** |
| 1926年8月 | 《彷徨》出版，赴厦门大学任国文系教授 | 第二部小说集问世，南下任教 |
| 1926年11月 | 写《范爱农》 | 怀念友人 |
| 1926年12月 | 从厦门大学辞职 | 不满校方环境 |
| 1927年1月 | 赴中山大学任教 | 继续南下 |
| 1927年3月 | 与中共两广区委书记陈延年会见 | 接触共产党人 |
| 1927年4月1日 | 赴黄埔军校发表演讲《革命时代的文学》 | 关注革命与文学的关系 |
| 1927年4月12日 | "四一二反革命政变"发生 | 目睹国民党的背叛 |
| 1927年4月29日 | 营救进步学生无果，愤然辞职 | 与国民党决裂 |
| 1927年8月 | 发表《魏晋风度及文章与药及酒之关系》 | 借古讽今的学术演讲 |
| 1927年9月 | 致信台静农，拒绝作为诺贝尔文学奖候选人；离开广州赴上海，与许广平开始同居生活 | 婉拒诺奖提名，开始上海时期 |
| 1927年12月 | 与梁实秋等人就"第三种人"、"自由人"发生论争 | 卷入重要文学论战 |

## 七、上海时期（1928-1936）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1928年春 | 参加中国革命互济会；与创造社、太阳社就"革命文学"问题展开论争；开始大量搜集马克思主义著作并翻译；提倡革命美术，倡导现代木刻运动 | 思想转向马克思主义，关注左翼文艺 |
| 1929年9月27日 | 许广平生子，取名"周海婴" | 晚年得子 |
| 1929年底 | 与冯雪峰多次磋商组建"中国左翼作家联盟" | 筹备左联 |
| 1930年2月 | 中国自由运动大同盟成立，为发起人之一 | 参与自由运动 |
| **1930年3月2日** | **出席中国左翼作家联盟成立大会，被选为常务委员，作《对左翼作家联盟的意见》演讲** | **成为左联领袖，左翼文学旗手** |
| 1931年1月20日 | 柔石被逮，鲁迅离寓避难；28日回旧寓 | 白色恐怖下的险境 |
| 1932年1月29日 | 遇战事（一二八事变），在火线中；次日避居内山书店；二月六日由内山书店友护送至英租界内山支店暂避 | 战火中的流亡 |
| 1932年 | 与艾青等人发起"春地美术研究所" | 支持青年艺术家 |
| 1933年1月 | 蔡元培函邀加入"中国民权保障同盟"，被举为执行委员 | 参与民权保障运动 |
| 1933年2月17日 | 蔡元培函邀赴宋庆龄宅，欢迎萧伯纳；作《为了忘却的纪念》怀念柔石 | 国际文化交流，悼念战友 |
| 1934年1月 | 与郑振铎合编的《北平笺谱》出版 | 古籍整理与出版 |
| 1934年5月 | 编序之木刻《引玉集》出版 | 支持新兴木刻运动 |
| 1935年2月 | 开始译果戈里《死魂灵》 | 晚年重要翻译工作 |
| 1935年6月 | 集成《新文学大系·小说二集》并作长序 | 总结新文学成就 |
| 1936年1月 | 肩及肋骨皆出现剧痛；最后的创新之作《故事新编》出版 | 重病中完成最后一部小说集 |
| 1936年2月 | 开始续译《死魂灵》第二部 | 带病工作 |
| 1936年5月15日 | 再发病，医生诊断胃疾，自后发热未愈 | 病情恶化 |
| 1936年5月31日 | 史沫特莱女士引美国邓医生来诊断，情况不乐观 | 确诊重病 |
| 1936年6月 | 身体略有好转，鲁迅及身边的人都认为"鲁迅先生好了" | 短暂的病情缓解 |
| 1936年6月9日 | 写《答托洛斯基派的信》 | 政治表态 |
| 1936年8月5日 | 写《答徐懋庸并关于抗日统一战线问题》 | 参与文艺界论争 |
| 1936年10月 | 写《因太炎先生而想起的二三事》 | 最后一篇文章 |
| 1936年10月17日 | 旧病复发 | 病情急剧恶化 |
| 1936年10月18日 | 黎明前疾病发作，气喘不止 | 生命最后时刻 |
| **1936年10月19日凌晨5时25分** | **在上海逝世** | **中国现代文学巨星的陨落，享年55岁** |

## 时间线总结

| 时期 | 时间跨度 | 核心特征 |
|------|----------|----------|
| 少年时期 | 1881-1898 | 家道中落，初识世态炎凉 |
| 南京求学 | 1898-1902 | 接触新学，接受进化论 |
| 日本留学 | 1902-1909 | **弃医从文**，确定人生方向 |
| 归国初期 | 1909-1917 | 沉默期，潜心古籍研究 |
| 新文化运动 | 1918-1923 | **《狂人日记》发表，现代文学奠基** |
| 创作高峰期 | 1924-1927 | **《呐喊》《彷徨》出版，民主战士** |
| 上海时期 | 1928-1936 | **左联盟主，马克思主义转向** |
